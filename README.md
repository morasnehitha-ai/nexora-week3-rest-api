NEXORA Week 3 - REST API Development

📌 Project Description

This project is a REST API built using Node.js and Express.js. It demonstrates CRUD (Create, Read, Update, Delete) operations, a basic User Login API, and a Contact Form API. The project was developed as part of the NEXORA UI Full Stack Development Internship - Week 3.

---

🚀 Features

- REST API using Express.js
- GET all products
- GET product by ID
- Add a new product
- Update an existing product
- Delete a product
- User Login API
- Contact Form Submission API
- JSON Request and Response Handling

---

🛠 Technologies Used

- Node.js
- Express.js
- JavaScript
- Postman (API Testing)

📡 API Endpoints

Home

GET /

Returns the welcome message.

Get All Products

GET /api/products

Returns all available products.

Get Product by ID

GET /api/products/:id

Returns a single product using its ID.

Add Product

POST /api/products

Creates a new product.

Update Product

PUT /api/products/:id

Updates an existing product.

Delete Product

DELETE /api/products/:id

Deletes a product.

User Login

POST /api/login

Validates user login credentials.

Contact Form

POST /api/contact

Accepts contact form details.
