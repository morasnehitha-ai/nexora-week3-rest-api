Week 3 - REST API Development

Description

A professional REST API built using Node.js and Express.js that demonstrates CRUD operations, user authentication, and contact form submission.

Features

- REST API using Express.js
- Product CRUD Operations
- User Registration API
- User Login API
- Get All Users API
- Contact Form Submission API
- JSON Responses
- Tested using Thunder Client/Postman

Technologies Used

- Node.js
- Express.js
- JavaScript
- REST API
- Thunder Client / Postman

API Endpoints

Products

- GET "/api/products"
- GET "/api/products/:id"
- POST "/api/products"
- PUT "/api/products/:id"
- DELETE "/api/products/:id"

Users

- POST "/api/register"
- POST "/api/login"
- GET "/api/users"

Contact

- POST "/api/contact"

How to Run

1. Install dependencies

npm install

2. Start the server

node server.js

3. Open

http://localhost:5000
